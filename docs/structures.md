These are users who gains the most from the platform.

```yaml

    type: string 
    auto_generated: true
name: 
    type: string
    required: true

    phone_number: 
        type: string
        required: true
        unique: true
    password: 
        type: string
        required: true
        hashed: true
    whatsapp_number: 
        type: string
    email: 
        type: string #email
gender: 
    type: enum #enum_gender
    required: true
dob: 
    type: date
    required: true

    type: string

    type:number

    type: enum # enum_states
    required: true

    required: true

    type: enum #enum_program

    type: string

    type: enum #enum_unit

    type: enum #enum

    type: enum #enum_yn
```

## R

### **Referral Job**
These are jobs posted by Referral partners. Seekers will be redirected to the Partner website for application submission.

```yaml
id: 
    type: string 
    auto_generated: true
partner_name: 
    type: string
    derived: FK to referral partner
job_title: 
    type: string
    required: true
job_company: 
    type: string
    required: true
job_location: 
    type: string
    required: true
job_description: 
    type: string
    addl: markdown_supported
date_of_posting: 
    type: date
    required: true
monthly_salary: 
    type: number
    required: true 
mode_of_work: 
    type: enum # enum_mode_of_work
    required: true
experience_required: # in years
    type:number
job_link: 
    type: url 
```
